# 🛒 Supermarket Sales Dashboard

An interactive **Streamlit dashboard** for analyzing supermarket sales data using **Python, Pandas, Plotly, and Streamlit**.

## 🚀 Live Dashboard

🔗 Add your deployed Streamlit app link here

## 📌 Project Overview

This dashboard helps analyze supermarket sales performance with interactive filters, KPIs, and visual insights.

## ✨ Features

* 📊 Sales Overview Dashboard
* 👥 Customer Insights Dashboard
* 📦 Product & Data View
* 📅 Date range filter
* 🏙️ City, branch, product line, and payment filters
* ⬇️ Download filtered data as CSV
* 📈 Interactive Plotly charts

## 📊 Key Metrics

* Total Sales
* Revenue
* Gross Income
* Quantity Sold
* Total Orders
* Average Sale
* Average Rating
* Top City
* Best Product Line

## 🛠️ Tools & Technologies

* Python
* Streamlit
* Pandas
* Plotly
* OpenPyXL

## 📷 Dashboard Pages

### 📊 Sales Overview

Shows sales KPIs, city-wise sales, daily sales trend, branch sales share, payment method analysis, and sales heatmap.

### 👥 Customer Insights

Shows customer count, customer revenue, gender-wise sales, customer type sales, ratings, and payment preferences.

### 📦 Product & Data View

Shows best product, top 5 products by revenue, quantity sold, product ratings, monthly revenue, and filtered dataset.

## 📌 Dataset

The dashboard uses supermarket sales data from an Excel file:

```text
supermarkt_sales.xlsx
```
